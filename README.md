# CNN-Breastcancer-Detection
Using a pre-trained CNN algorithm to analyze breast mammogram images. 

## Requirements
- numpy==1.21.2
- Pillow==8.4.0
- streamlit==0.79.0 
- tensorflow==2.4.0

## Dataset
we collect the data from Kaggle - (https://www.kaggle.com/kmader/mias-mammography)
